## **1.20.O Error taxonomy (client‑safe)**

- *UPLOAD_FAILED*, *UNSUPPORTED_TYPE*, *FILE_TOO_LARGE*, *PUSH_DENIED*, *SAFE_MODE_BLOCKED*, *AUTH_EXPIRED*, *OFFLINE_RETRYING*.
- Each error includes user‑friendly copy and retry guidance.
